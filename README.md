# Android Learning Projects

This repository contains various Android learning projects and examples.

## Structure
```
android/
├── 02hellokitty/          # Basic Hello World Android app
├── [future projects]/     # Additional learning projects
└── .gitignore            # Android Studio gitignore configuration
```

## Projects

### 02hellokitty
- **Type**: Basic Android Application
- **Language**: Kotlin
- **SDK**: Compile SDK 36, Min SDK 23
- **Description**: Simple Hello World application with modern Android practices

## Getting Started

Each project is self-contained and can be opened directly in Android Studio:

1. Clone this repository
2. Open Android Studio
3. Choose "Open an existing project"
4. Navigate to the specific project folder (e.g., `02hellokitty`)

## Requirements

- Android Studio Hedgehog | 2023.1.1 or later
- Android SDK API 36
- Kotlin 2.0.21

## Repository Configuration

This repository is configured with a comprehensive `.gitignore` file that excludes:
- Build artifacts (*.apk, *.dex, build/, .gradle/)
- IDE files (*.iml, .idea/)
- Local configuration files
- OS-specific files
- Keystore files (for security)

This ensures clean version control across multiple Android projects.
